#### java相关书籍

1. ##### java8函数式编程

   ​     多年以来，函数式编程被认为是少数人的游戏，不适合推广给普罗大众。写作此书的目的就是为了挑战这种思想。本书将探讨如何编写出简单、干净、易读的代码；如何简单地使用并行计算提高性能；如何准确地为问题建模，并且开发出更好的领域特定语言；如何写出不易出错，并且更简单的并发代码；如何测试和调试Lambda表达式。

   ​      如果你已经掌握Java SE，想尽快了解Java 8新特性，写出简单干净的代码，那么本书不容错过。
